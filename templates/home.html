API for Predicting prices in real estate

The following endpoints are available on this server (http://0.0.0.0:5041).

MethodRouteDescription
GET/This home page
POST/predictPredict a house sale price from JSON input
GET/pipelineOverview of the data science pipeline

Curl predict example:

curl -X POST http://localhost:5041/predict \
  -H "Content-Type: application/json" \
  -d '{"OverallQual":8,"GrLivArea":2000,"GarageCars":2,"TotalBsmtSF":1000,"1stFlrSF":1000,"2ndFlrSF":800}'
